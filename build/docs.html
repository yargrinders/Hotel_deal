Техническое задание:

НАШ МАКЕТ

  1. Установить и развернуть Проэкт "Hotel_deal" в "Раздел 1"
  2. Подготовить рабочую среду для работы в команде "Раздел 2
  3. Ознакомиться с структурой сборки в "Раздел 3"
  4. Выполнить все пункты в "Раздел 4"
  5. Залить готовую ветку "Раздел 5"

Раздел 1

Содержимое раздела 1.

Все действия происходят в Vscode в терминале

  1. git clone https://github.com/yargrinders/Hotel_deal.git
  2. cd Hotel_deal
  3. npm i
  4. clear
Далее по списку 2/Раздел 2

Раздел 2

Содержимое раздела 2.

Все действия происходят в Vscode в терминале. Так как мы находимся в ветке main, а она является основной. Вносить в ней какие либо изменения - ЗАПРЕЩЕННО !!! Для этого мы должны слздать её копию (другую ветку) как это сделать - ниже по инструкции

  1. Список Всех основных команд Git -> тут ССЫЛКА на команды / Helper
  2. git checkout -b Burger
    3. так как ты будешь заниматься бургер меню
  4. Далее мы можем вносить свои изменения в код по инструкциям дальше в ветку Burger
  5. После Внесения изменений в ветку Burger ее нужно залить на репозиторий / пошаговые комманды ниже
    1. git add .
      2. Индексируйте ваши изменения
    2. git commit -m 'Комментарий к ветке Burger'
      3. Создайте коммит (example Burger v.1.2)
    3. git push origin Burger
      4. И отправьте изменения на удаленный сервер

Раздел 3

Содержимое раздела 3.

Моя сборка gulp простая и понятая.

  1. коммпанда "gulp" - запускает сборку в режиме разработчка. Создается папка "build"
  2. комманда "gulp docs" - запускает сборку в режим сдачи проэкта. Создается папка "docs" - это то что мы должны отдать заказчику - Это комманда - нужна для финальной тестировки макета
  3. папка src - это рабочая - в ней все для работы

По моей структуре - много чего запутанно но постараюсь в двух словах.

для выполнения пунктов из Раздела 4 - тебе понадобаться работа со следующими файлами

  1. html/blocks/header.html - Хедер инклюдит нав
  2. html/blocks/_nav.html - вся навигация
  3. scss/blocks/_header.scss стили думаю найдешь

ознакомься с скрином
